# 7even Mobile

React Native (Expo) client for 7even MVP.

## Current MVP Screens
- Login
- Register
- Quiz onboarding
- Current match + top 5 suggestions
- 1:1 realtime chat

## Start

1. Install deps at repo root: `npm install`
2. Start app: `npm run dev:mobile`
3. Ensure API is running at `http://localhost:4000`
